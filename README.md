# RUN.game developer platform

RUN.game simplifies HTML5 game development and publishing. It gives you an SDK that handles many advanced features, saving you development time, and simplifies publishing, so that you can focus on what matters most: building the next big thing in gaming.

Any genre or style of game can be made for RUN.game, as long as it's built with HTML5.

To get started making games on RUN.game, you will need to do the following:

* Install the RUN.game CLI
* Install the RUN.game SDK
