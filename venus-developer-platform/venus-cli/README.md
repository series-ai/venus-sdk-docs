The RUN.game CLI ([command-line interface](https://en.wikipedia.org/wiki/Command-line_interface)) is what you'll use to initialize a new game, administer your game, and deploy your game to the RUN.game platform.

[How to install the CLI.](installing-the-venus-cli.md)

`rundot --help` shows usage information.

All `rundot` commands have a `--help` option as well.
