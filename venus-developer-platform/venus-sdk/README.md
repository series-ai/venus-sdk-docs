# Venus SDK

## What is the Venus SDK?

The Venus SDK is a collection functionalities that you'll be able to leverage from to get your in game to use them!

The Venus SDK is designed for LLMs to easily utilize and write code for it. All the human operator needs to be aware of here is what functionality exists and how they can leverage it to enhance their game.

{% hint style="info" %}
Check how to Install/Import the SDK here: [quickstart.md](quickstart.md "mention")
{% endhint %}

## SDK Features

These features include:

* Loading screen customization
* Monetization features (IAP & Ads)
* Haptics feedback
* Local notifications
* Manage and implement save states
* Game Analytics
* LLMs endpoints&#x20;
