# Quickstart

We offer the  SDK as a convenient npm package.

{% stepper %}
{% step %}
### Install Node.js

<a href="https://nodejs.org/en/download" class="button primary">Download and install Node.js</a>

We support Node.js 20.0 or higher.
{% endstep %}

{% step %}
### Install the SDK with npm

From the root folder of your project:

```shell
npm install @series-inc/rundot-game-sdk@latest
```

{% hint style="info" %}
The SDK is updated frequently. To update it in your project, simply re-run the above command.
{% endhint %}
{% endstep %}
{% endstepper %}
